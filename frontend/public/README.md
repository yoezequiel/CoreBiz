## Open Graph Image

Para generar la imagen Open Graph (`og-image.png`), puedes usar cualquiera de estas opciones:

### Opción 1: Canva (Recomendado)

1. Ir a [Canva.com](https://www.canva.com)
2. Crear diseño personalizado de **1200 x 630 px**
3. Usar plantilla "Open Graph" o crear desde cero
4. Elementos sugeridos:
    - Fondo azul (#2563eb) o degradado
    - Logo/Texto "CoreBiz ERP"
    - Subtítulo: "Sistema Multi-Tenant de Gestión Empresarial"
    - Iconos: gráficos, dashboard, analytics
5. Descargar como PNG y reemplazar `public/og-image.png`

### Opción 2: Figma

1. Crear frame de 1200 x 630 px
2. Diseñar con elementos del proyecto
3. Exportar como PNG 2x

### Opción 3: Generadores Online

- [OG Image Playground](https://og-playground.vercel.app/)
- [Social Image](https://social-image.vercel.app/)

### Especificaciones Técnicas

- **Dimensiones**: 1200 x 630 px (ratio 1.91:1)
- **Formato**: PNG o JPG
- **Tamaño**: Máximo 8 MB (idealmente < 1 MB)
- **Colores**: Usar la paleta del proyecto (#2563eb como principal)
